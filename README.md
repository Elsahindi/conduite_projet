# Application of mutual assistance
## Description
This **Maven** project implements a *chat system* that allows interaction between users. The project is designed to become a *Vulnerable Person Help application* that offers help to people in need. Individuals can enter their needs and volunteers who sign up can offer help. A validator validates the different missions.

## Prerequisites
Before you begin, make sure you have the following installed on your machine:
- **JDK 11**
- **Maven**

## Installation
1. Clone this repository to your local machine:

`git clone https://github.com/insa-4ir-chatsystem/chatsystem-hamdan-da-costa-bento.git](https://github.com/Elsahindi/conduite_projet.git`

2. Navigate to the project directory:

`cd conduite_projet`

3. Manage dependencies with Maven:

`mvn install`

`mvn compile`

## Usage
To run the application, execute the main class via Maven:

`mvn exec:java -Dexec.mainClass="Main"`

## Tests
The project includes unit tests that can be run via Maven:

`mvn test`
